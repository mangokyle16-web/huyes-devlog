# Huyes 桌上型分豆機 — 專案 Roadmap
**更新日期：2026-05-31**

---

## 產品目標

桌上型消費級多光譜分豆機（參考外觀：`CoffeeBeanSorter.png`）

| 規格 | 目標 |
|------|------|
| 吞吐量 | ~1 顆/秒，300g（約 900 顆）≤ 15 分鐘 |
| 核心分析 | Agtron 烘焙值 / 黴菌偵測 / 光譜指紋 / 計數 |
| 輸出 | 良豆桶 / 瑕疵桶自動分流 |
| 平台 | Raspberry Pi 5 + CM020D 多光譜相機 + 7" DSI 觸控螢幕 |
