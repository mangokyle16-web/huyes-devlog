# 提案 06：手機遠端採集指揮台 — 隨時掌控 Pi5

> 由 Mac Mini Claude Code 在 Kyle 休息時自主思考產生（2026-06-06 凌晨）

---

## 一句話定義

一個輕量 Web 介面，讓你在任何設備（手機、平板、其他電腦）都能遠端啟動/暫停採集、查看即時 bean 統計、設定採集參數——不需要 SSH，不需要打開 terminal。

---

## 為什麼需要

今天（2026-06-05）Kyle 問：「我要怎麼用手機遠端控制？」

目前採集 pipeline 完全靠 SSH + command line 操作。要啟動採集需要：
1. 開啟 terminal
2. SSH 到 Pi5
3. 輸入完整指令

對於日常採集（Kyle 只是要開始拍豆子），這個流程太重。

**理想情況：**
- 拿起手機 → 開瀏覽器 → tap 一個按鈕 → 開始採集
- 不需要 SSH，不需要 terminal，任何人都能操作

## 技術實作

### 後端（Pi5，FastAPI）

```python
# spectral_capture/control_server.py
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles
import subprocess, sqlite3

app = FastAPI()

pipeline_proc = None

@app.post("/capture/start")
def start_capture(origin: str = "unknown", roast: str = "green", interval: int = 30):
    global pipeline_proc
    if pipeline_proc and pipeline_proc.poll() is None:
        return {"status": "already_running"}
    pipeline_proc = subprocess.Popen(
        ["python3", "-u", "spectral_capture/capture_pipeline.py",
         "--origin", origin, "--roast", roast, "--interval", str(interval)],
        stdout=open("/tmp/pipeline.log", "w"),
        stderr=subprocess.STDOUT
    )
    return {"status": "started", "pid": pipeline_proc.pid}

@app.post("/capture/stop")
def stop_capture():
    if pipeline_proc:
        pipeline_proc.terminate()
    return {"status": "stopped"}

@app.get("/capture/status")
def status():
    running = pipeline_proc and pipeline_proc.poll() is None
    log = open("/tmp/pipeline.log").readlines()[-10:] if running else []
    conn = sqlite3.connect("spectral_capture/data/beans.db")
    count = conn.execute("SELECT COUNT(*) FROM bean_spectra").fetchone()[0]
    conn.close()
    return {"running": running, "total_beans": count, "recent_log": log}
```

### 前端（單一 index.html，無 framework）

- 純 HTML + CSS + Vanilla JS
- 每 3 秒 `GET /capture/status` 更新畫面
- 適配手機螢幕（max-width: 420px）
- 深色主題（夜晚操作不刺眼）

### 啟動方式（Pi5）

```bash
cd ~/KyleClaude
uvicorn spectral_capture.control_server:app --host 0.0.0.0 --port 8765
```

手機連同一 WiFi → 瀏覽器開 `http://raspberrypi.local:8765`

### 外網存取（可選）

若需要不在家時遠端操作，可加上 Cloudflare Tunnel：
```bash
cloudflared tunnel --url http://localhost:8765
```
會產生一個 `*.trycloudflare.com` URL，手機任何地方都能連。
