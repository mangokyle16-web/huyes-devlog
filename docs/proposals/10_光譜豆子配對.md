# 提案 10：光譜-豆子配對 — 每顆豆子的個別光譜向量

> 由 Mac Mini Claude Code 在 Kyle 休息時自主思考產生（2026-06-12 凌晨）

---

## 背景

現有光譜採集 pipeline 紀錄「per-frame 平均光譜」。
YOLOX-tiny bbox 精準後，可以升級為「per-bean 個別光譜」。

## 提案

同一幀圖片，對每個 YOLOX bbox 分別截 ROI，計算各自的多光譜特徵向量：

```
frame_000030.qs + YOLOX detections →
  bean_0: bbox(406,570,159,203) → 5波段 ROI 平均 → [b450, b560, b650, b730, b840]
  bean_1: bbox(249,617,195,150) → 5波段 ROI 平均 → [b450, b560, b650, b730, b840]
  ...
```

## DB Schema 升級

從 `bean_spectra`（per-frame）升級為 per-bean：

```sql
CREATE TABLE bean_records (
    id          INTEGER PRIMARY KEY,
    frame_id    INTEGER NOT NULL,
    captured_at REAL    NOT NULL,
    bbox_x      INTEGER, bbox_y INTEGER, bbox_w INTEGER, bbox_h INTEGER,
    score       REAL,                          -- YOLOX confidence
    b450, b560, b650, b730, b840  REAL,        -- 5-band spectrum
    label       TEXT DEFAULT 'unknown'          -- 未來瑕疵分類標籤
);
```

## 效益

- 不需要額外標注，直接從 YOLOX bbox 建立 per-bean 光譜資料集
- 這是 Siamese 多光譜瑕疵偵測（提案 07）的直接前置資料基礎
- 累計 500+ 顆 per-bean 記錄後，可以開始訓練 Siamese 網路
